BICC Pipeline Skeleton — pkg_bicc_common
=========================================

This package provides shared utilities for the BICC (BI Cloud Connector)
CSV-to-Oracle pipeline. It is used by all entity-specific packages
generated by the Fusion Data Pipeline Generator.

Prerequisites
-------------
1. Oracle Autonomous Database (ATP/ADW) with APEX installed
2. OCI Object Storage bucket for BICC extract files
3. DBMS_CLOUD credential for Object Storage access:

   BEGIN
       DBMS_CLOUD.CREATE_CREDENTIAL(
           credential_name => 'YOUR_OBJ_STORE_CREDENTIAL',
           username        => '<oci_user_ocid>',
           password        => '<auth_token>'
       );
   END;
   /

4. Supporting tables — DDL scripts are included in this ZIP:
   @bicc_loader_map.sql  — maps FILE_LIKE patterns to load types (create first)
   @bicc_files.sql       — tracks discovered BICC ZIP files + discovery view
   @bicc_load_job.sql    — job tracking table
   @bicc_load_log.sql    — load audit log

   Run these scripts BEFORE compiling the package.
   Then update the credential and bucket URI in bicc_files.sql (the view).

Setup
-----
1. Run the four supporting table scripts (see above)

2. Update the two constants in pkg_bicc_common.sql:
   - gc_credential  => your DBMS_CLOUD credential name
   - gc_bucket_uri  => your Object Storage bucket URI

3. Compile the spec first, then the body:
   @pkg_bicc_common.sql
   @pkg_bicc_common.plb

4. Use the Pipeline Generator to create entity-specific packages.
   Each generated package depends on pkg_bicc_common.

What's Included
---------------
- safe_to_number()        — tolerant NUMBER conversion
- safe_to_timestamp()     — handles MM/DD/YYYY, YYYY-MM-DD, ISO 8601
- extract_and_stage_csv() — downloads ZIP, extracts CSV, uploads to staging
- refresh_bicc_files()    — syncs file list from Object Storage
- purge_bicc_objectstore()— deletes old ZIP files past retention
- purge_copy_tables()     — drops COPY$ housekeeping tables
- run_bicc_daily_today()  — orchestration stub (customize per project)
